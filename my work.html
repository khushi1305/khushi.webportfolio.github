My Work
1. Lyrics Grabber

As a part of my diploma course, I'd done this project based on music, which is done using Html, Css, Bootstrap as front-end and Php and MySql as back-end.

This project consists of the development of a contributory website to display lyrics of the bollywood songs. The goal of this website is to allow users to sing along with the lyrics. There are many such applications available in the market, but it needs to be downloaded and requires storage space. Here, user can find all kinds of music genres and play music according to the mood. LYRICS GRABBER incorporates premium user facility, which provides a high quality audio with downloading facility. This website allows user to search for their favorite songs according to song name, artist, album and also create a playlist.

It is a completely dynamic website in which all the songs, albums, artists, playlist and everything comes from the database linked with it. Some pictures of the front page are displayed below...
